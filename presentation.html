Multi-Model Routing

Every message gets classified and sent to the best AI for the job.

GPT-4oReasoning, planning, tool use
Claude CodeEngineering, coding, debugging
GeminiLarge documents, research
KimiReports, slides, spreadsheets
GPT-4o-miniSimple Q&A, formatting (cheap)

The router uses keyword detection + GPT-4o-mini classification to decide the route. Zero config needed.

Examples:

  • "Fix the login bug" → Claude Code
  • "Analyze this PDF" → Gemini
  • "Make me a report" → Kimi
  • "What's up?" → GPT-4o (chat)

Teach Mode — make it yours

Teach your bot custom commands that run deterministically — same trigger, same steps, every time. Zero API cost to execute.

# Teaching a command
"teach: when I say 'deploy',
 pull from git, install deps,
 and restart the bot"

# Using it later
"deploy"
✓ git pull
✓ npm install
✓ pm2 restart

# Managing commands
"my commands"    # list all
"delete command #3"  # remove

Why it's powerful

  • The more you teach it, the more personalized it becomes
  • Pipelines run tool-by-tool — no AI hallucination
  • Can chain any tool: memory, server, browser, email, vault
  • Commands are per-user in group chats
  • Track usage stats (execution count, last run)

The bot gets smarter the more you use it — without costing more.

Getting your API Keys

OpenAI (Required)

  • Go to platform.openai.com → Sign up or log in
  • Click your profile (top-right) → API Keys
  • Click Create new secret key → copy it (starts with sk-)
  • Go to Billing → add a payment method → set a usage limit ($10-20/mo is plenty)
  • You must have credits loaded. A new account with $0 balance will get 429 errors

Gemini (Free)

  • Go to aistudio.google.com/apikey
  • Sign in with Google → click Create API Key
  • Copy the key — that's it, no billing needed

Brave Search (Free tier)

  • Go to brave.com/search/api
  • Sign up → choose the Free plan (2,000 queries/mo)
  • Copy your API key from the dashboard

Troubleshooting

ProblemFix
Bot not responding Run pm2 status. If stopped: pm2 restart favor. Check logs: pm2 logs favor --lines 30
QR code won't scan Make sure WhatsApp is updated. Try ./relink.sh to generate a new QR code
WhatsApp disconnects often Normal for first 24hrs. Bot auto-reconnects. If persistent: pm2 restart favor
"429 Too Many Requests" OpenAI rate limit. Add billing to your OpenAI account and load credits ($5-10)
Bot replies are slow Normal for first message (cold start). Subsequent replies are faster. GPT-4o is slower than mini
"Cannot find module" Run npm install in the favor directory. Then pm2 restart favor
Config changes not working Config hot-reloads every 2 seconds. Or send /reload on WhatsApp. Or pm2 restart favor
Chromium/Puppeteer errors Run sudo apt install -y chromium-browser (Linux) or brew install chromium (Mac)
Out of memory Add swap: sudo fallocate -l 2G /swapfile && sudo chmod 600 /swapfile && sudo mkswap /swapfile && sudo swapon /swapfile

Tips & Best Practices

Save Money

  • Set a spend cap in Guardian config (maxDailySpend)
  • Use /model gpt-4o-mini for casual chat (10x cheaper)
  • Gemini is free — use it for large document analysis
  • Teach Mode commands cost $0 to execute
  • Set OpenAI billing alerts at $10 and $20

Stay Secure

  • Never share your config.json — it has your API keys
  • Guardian auto-redacts leaked keys, but don't test it
  • Set the security phrase in config for vault access
  • Keep your server updated: sudo apt update && upgrade
  • The bot only responds to your phone number by default

Get the Most Out of It

  • Write detailed knowledge files — the better your identity.md, the better the bot
  • Use memory actively: "remember that..." builds its knowledge
  • Set up the Alive Engine for daily check-ins
  • Teach it shortcuts for things you repeat
  • Try Build Mode — it can literally code for you
